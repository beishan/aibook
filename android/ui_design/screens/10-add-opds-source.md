# 10 添加 OPDS 数据源

参考图：`../images/10-add-opds-source.png`

## 表单字段
- 数据源名称（必填）
- 服务器 URL（必填，http/https 校验）
- 用户名（可选）
- 密码（可选，隐藏显示按钮）
- 同步模式：完全同步 / 增量同步
- 测试连接按钮

## 校验与行为
- URL 需明确是 OPDS 目录地址，不是普通网页地址。
- 点击测试连接后展示：版本、认证结果、找到的入口数/书籍数、错误原因。
- 仅在测试成功后允许“保存并启用”，也可允许用户保存为禁用状态。
- 密码不回显；编辑时只显示“已保存”，允许更新或清除。
