# 05 OPDS 数据源管理

参考图：`../images/05-opds-sources.png`

## 功能
管理多个 OPDS 数据源：启停、同步、编辑、删除、查看源目录、查看错误。

## 卡片字段
- 图标、源名称、地址（可截断）、状态点、最近同步时间、书籍数量。
- 操作：立即同步、编辑、更多菜单、启用开关。

## 行为
- 同步状态：IDLE/SYNCING/SUCCESS/FAILED；失败时展示人类可读消息与“重试”。
- 禁用数据源后：书城默认不再展示其目录；已下载或已加入书架的书仍保留，并明确标识来源已禁用。
- 删除数据源：不自动删除已下载文件，确认弹窗提供单独复选框。
