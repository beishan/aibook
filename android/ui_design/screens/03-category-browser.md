# 03 分类查找 / 全部浏览

参考图：`../images/03-category-browser.png`

## 筛选维度
- 来源：全部 / 本地 / OPDS
- 分类：全部、小说、文学、社科、历史、科技、外文、少儿……
- 排序：综合、评分、最新、标题
- 可选过滤：格式、是否已下载、是否已加入书架、指定 OPDS 数据源

## 布局
- 筛选项使用横向可滚动 Chips。
- 书籍使用三列网格（大屏）或双列网格（普通屏）；内容较长时标题最多两行，作者一行省略。
- 来源标签在封面底部或信息区显著显示。

## 行为
- 筛选条件需保存在 `SavedStateHandle`，返回书城后保持。
- 滚动分页：本地通过数据库分页，OPDS 通过下一页链接/游标。
- 空态明确展示当前筛选条件，并提供“清除筛选”。
