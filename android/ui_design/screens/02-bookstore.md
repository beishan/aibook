# 02 书城（全量聚合浏览）

参考图：`../images/02-bookstore.png`

## 页面目的
聚合本地已导入书籍和已启用 OPDS 数据源，提供搜索、来源筛选、分类入口、推荐区和全部浏览。

## 顶部与筛选
- 顶部：标题“书城”、搜索按钮、排序/管理入口。
- 来源分段：全部 / 本地 / OPDS。
- 搜索框：可搜索书名、作者、分类。
- 分类 Chips：文学、科幻、历史、经管、心理、传记等；由本地和 OPDS 分类聚合而来。

## 内容
- 推荐区：可由“最近导入”“最近更新”“随机探索”“用户收藏类别”组成；没有算法时不可伪造个性化。
- 全部浏览：卡片展示封面、标题、作者、来源标签、评分（可选）和更多菜单。
- 最近更新：基于 OPDS 同步时间和本地导入时间。

## 行为
- 点击书籍：`book/{bookId}`。
- 点击“本地/OPDS”：立即过滤，不清空其他分类条件。
- 点击 OPDS 标签：可进入源目录或以该源筛选。
