# 09 扫描目录管理

参考图：`../images/09-scan-directories.png`

## 内容
- 顶部快捷卡：立即扫描 / 添加目录。
- 自动扫描开关：应用启动后扫描已启用目录；实际实现应由 WorkManager/节流策略控制，避免每次启动都长时间占用资源。
- 已配置目录列表：显示目录名称、最近扫描时间、发现数量、开关、更多菜单。
- 提示支持 EPUB/TXT/PDF 等。

## 行为
- 添加目录使用 `ACTION_OPEN_DOCUMENT_TREE`，保存 URI 持久权限。
- 单个目录菜单：立即扫描、重命名显示名、清除扫描记录、禁用、移除目录。
- 扫描过程显示进度，用户退出页面后仍继续（前台服务/WorkManager 视文件量决定）。
