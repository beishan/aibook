# 08 书籍详情

参考图：`../images/08-book-detail.png`

## 信息
- 封面、书名、作者、来源（本地/OPDS）、格式、评分（可选）、标签。
- 简介、书籍信息（文件大小、字数、添加时间、最后阅读）。
- 相关书籍/同作者/同分类。

## 主操作（按状态变化）
- 已下载且未在书架：加入书架 / 开始阅读。
- 已在书架：继续阅读 / 移出书架。
- OPDS 在线书：下载 / 加入书架（若源支持可自动下载）。
- 本地文件：开始阅读、加入书架、定位文件、删除本地文件。

## 规则
- 加入书架要有轻量反馈（Snackbar）。
- 文件不存在或下载失败时禁用阅读按钮，但提供修复入口。
