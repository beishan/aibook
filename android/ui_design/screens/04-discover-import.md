# 04 发现：导入与本地扫描

参考图：`../images/04-discover-import.png`

## 页面目的
将“主动导入文件”和“配置扫描目录”放在一个明显的工具入口，而非塞进书架。

## 内容
- 首屏两张大功能卡：
  - 立即导入文件：打开系统文件选择器，可多选 EPUB/TXT/PDF；导入后解析元数据并进入书城本地范围。
  - 扫描本地目录：进入扫描目录管理。
- 已配置扫描目录摘要：目录名称、开关、上次扫描时间、发现书籍数量。
- 最近导入：封面、文件名、格式、解析状态、是否已加入书架。
- 底部提示：支持格式及加密/损坏文件的处理提示。

## 行为
- 导入成功不自动加入书架；成功页给“加入书架”快捷操作。
- 重复书识别：提示“可能已存在”，让用户选择保留两个版本/替换元数据/取消。
