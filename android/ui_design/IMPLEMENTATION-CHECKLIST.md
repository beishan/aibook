# 开发验收清单

## 书架
- [ ] 只展示 `isInShelf = true` 的书。
- [ ] 支持继续阅读、移出书架、批量管理。
- [ ] 书籍文件丢失有可修复状态。

## 书城
- [ ] 聚合本地与启用的 OPDS 数据源。
- [ ] 来源、分类、格式、排序可组合筛选。
- [ ] 搜索在离线时仍返回本地结果。

## 本地导入与扫描
- [ ] `ACTION_OPEN_DOCUMENT` 多选导入。
- [ ] `ACTION_OPEN_DOCUMENT_TREE` 保存目录持久授权。
- [ ] 扫描不会自动把所有文件加入书架。
- [ ] 重复书有用户可控处理。

## OPDS
- [ ] 多数据源、启停、编辑、同步、错误重试。
- [ ] 认证信息加密保存。
- [ ] 已下载书在源禁用/删除后可继续阅读。

## 阅读器
- [ ] 进度可恢复。
- [ ] 目录、主题、字号、行距、翻页方式可用。
- [ ] 书签/笔记/划线可创建、定位、导出。

## 设置与质量
- [ ] 页面主题与阅读主题分离。
- [ ] 清理缓存不会误删用户书籍和笔记。
- [ ] 权限用途清晰可解释。
- [ ] 暗色/护眼主题文字对比度合格。
