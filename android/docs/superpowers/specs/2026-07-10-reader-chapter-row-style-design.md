# 阅读器分卷目录章节条目样式设计

## 目标

只调整 Android 阅读器分卷目录中的章节条目，使其接近参考图的轻量列表样式。卷标题、折叠交互、目录顶部、经典目录、阅读设置和其它页面保持不变。

## 章节条目

- 章节行保持 68dp 高度。
- 移除现有两位数字列，直接显示章节的完整标题。
- 右侧显示现有阅读状态文案，并增加向右箭头。
- 普通章节使用白色背景，不再显示整圈描边。
- 普通章节之间使用低透明度的细底部分割线；分卷最后一章不显示分割线。
- 点击章节继续沿用现有跳转和关闭目录行为。
- 点击反馈只使用颜色或透明度变化，不使用阴影或按压投影。

## 当前章节

- 当前章节使用 14dp 圆角浅强调色背景。
- 左侧显示约 4×28dp 的圆角短竖条，不再使用贯穿整行的色条。
- 当前章节的标题、状态文案和右箭头使用强调色。
- 阅读状态文案保持现有“当前阅读”，本次不改业务文案。

## 约束

- 只修改 `GroupedChapterRow` 及其直接需要的章节条目展示数据。
- 不修改 `GroupedContentsHeader`、分卷识别、展开状态、惰性列表结构或设置持久化。
- 不改变经典目录的 `ChapterRow`。
- 不修改全局设计令牌或其它页面。

## 测试与验证

- 先为章节条目展示数据补充失败测试，覆盖当前章节状态和末行分割标记。
- 实现最小展示模型和 Compose 样式后运行相关单元测试。
- 运行完整相关模块测试和 Debug APK 构建。
- 使用 `git diff --check` 并审计最终变更范围只包含章节条目实现、对应测试和本规格文档。
