# 阅读器完整顶部栏设计

## 目标

阅读内容区域被点击后，显示参考图风格的顶部阅读栏。书签操作移至顶部栏，底部原书签入口替换为亮色/暗色阅读主题的快捷切换；补齐当前书籍的正文搜索。

## 布局与交互

- 顶部栏覆盖在阅读内容顶部，使用当前阅读主题的背景与细分隔线，不使用阴影。
- 左侧为返回；中间两行显示书名与当前章节名，过长时省略；右侧依次为搜索、书签和更多。
- 书签沿用当前读书位置的新增/删除逻辑，并以填充图标和强调色表示已添加。
- “更多”仅显示三点图标且无点击行为，作为后续入口占位。
- 搜索打开阅读内搜索栏；输入关键字后在已解析的章节文本中匹配，显示命中数量，支持上一处/下一处跳转及关闭。跳转到命中位置后关闭菜单以回到阅读。
- 底部末项为亮/暗主题快捷按钮：当前为 `DARK` 时显示“亮色”，点击切到 `LIGHT`；其余阅读主题均显示“暗色”，点击切到 `DARK`。仅在 `LIGHT` 和 `DARK` 间切换。
- 按钮点击态仅使用系统无阴影水波或颜色状态，不添加按压投影。

## 数据与边界

- 搜索索引由现有 `ReaderUiState.chapters` 的标题和正文生成，匹配大小写不敏感。
- 每一个命中记录章节索引与章节内行索引；跳转复用 `ReaderViewModel` 的章节选择与页/滚动定位能力。
- 远程内容没有本地书签持久化对象，因此顶部书签沿用现有行为（无操作）；搜索仍可用。
- 空关键字或无匹配时，不执行跳转并在界面显示明确状态。
