- Pinia store 统一使用 Composition API 风格的 `defineStore('id', () => { ... })`，state 通过 `ref` 声明并在函数体内返回。
- 所有 HTTP 请求集中通过 `@/utils/api` 导出的 axios 实例发起，禁止在组件中直接 new axios 实例。
- 路由定义采用懒加载 `() => import('@/views/...')` 形式，并通过 `meta.requiresAuth` 配合 `router.beforeEach` 做鉴权控制。
- 组件与模块路径统一使用 `@/` 别名进行绝对导入，避免相对路径层级混乱。
- 用户认证凭据统一存放在 `localStorage.token`，store 与 axios 拦截器双向读写该 key 完成登录态维护。
