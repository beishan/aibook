- 解析器统一暴露为顶层 `object` 并提供静态 `parse(...)` 工厂方法，内部状态全部私有，保证无副作用。
- 所有公开数据类采用 Kotlin `data class`，字段使用不可变 `val`，默认值用于可选元数据（如 `EpubMetadata`、`ReaderBookmark.createdAt`）。
- XML 解析一律通过 `DocumentBuilderFactory.newInstance().apply { isNamespaceAware = true; setFeature(disallow-doctype-decl=true); ... }` 构造，显式关闭外部实体与 DOCTYPE 以防御 XXE。
- DOM 节点遍历使用 `getElementsByTagNameNS("*", name)` 配合 `Element.firstTextByLocalName` 扩展，避免硬编码命名空间前缀。
