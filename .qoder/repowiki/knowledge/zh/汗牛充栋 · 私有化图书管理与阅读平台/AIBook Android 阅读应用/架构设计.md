采用 Gradle 多模块分层架构：
- `app` 为唯一可安装产物，负责 `AiBookApplication` 启动、`MainActivity` 入口与 Hilt DI（`ServiceLocator.kt`），并通过 `feature/*` 下的 Screen/ViewModel 组合导航。
- 依赖自底向上的纯 Kotlin JVM 库：`core_model` 定义领域模型；`core_reader` 提供 EPUB/文本解析与分页；`core_network` 封装 Retrofit + OkHttp 的 REST API 与 OPDS 目录服务；`core_data` 基于 Room + DataStore 实现持久化并向上暴露 Repository 接口。
- 各 feature 通过 Hilt 注入对应 Repository，Repository 再组合 network/data 子模块能力，形成单向依赖链：app → feature → repository → (core_network | core_data) → core_model/core_reader。
- 主题与设计系统集中在 `ui/theme` 与 `ui/design`，被所有 feature 复用。
